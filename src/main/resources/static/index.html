📋 테스트 방법

1단계: 위의 "카카오 로그인 시작하기" 버튼 클릭

2단계: 카카오 로그인 페이지에서 로그인

3단계: 동의 항목 확인 후 동의

4단계: 결과 확인 (토큰 또는 에러 메시지)

🔗 테스트 엔드포인트

GET /oauth2/authorization/kakao
GET /oauth2/kakao

⚙️ 설정 확인사항

카카오 개발자 콘솔에서 Redirect URI가 다음과 같이 설정되어 있는지 확인하세요:

http://localhost:8080/login/oauth2/code/kakao
